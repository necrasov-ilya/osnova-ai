# MVP-архитектура приложения

Этот MVP должен уже открываться на телефоне, распознавать текст через камеру, прогонять результат через Gemma и добавлять черновик в заметку. Без идеальной VLM-магии, но с рабочим контуром.

## Что входит в первый MVP

- Kotlin Android-приложение.
- Главный экран заметок.
- Создание заметки.
- Экран заметки с нижней пилюлей.
- Раскрывающийся камерный контейнер.
- Проверка и скачивание OCR-модуля при первом запуске.
- Проверка и скачивание Gemma 4 E2B `.litertlm` под Snapdragon 8 Elite.
- CameraX preview.
- ML Kit Text Recognition как первый рабочий распознаватель.
- LiteRT-LM runtime dependency.
- Попытка запуска Gemma и сборки блока конспекта из OCR-текста.
- Очередь обработки.
- Автоматическая вставка распознанного текста в заметку.
- Две темы с одинаковым акцентом и разным фоном.

## Что пока не входит

- Полноценный VLM-анализ изображения без OCR-промежуточного слоя.
- Надёжная настройка всех backend LiteRT-LM под разные SoC.
- Умный LLM Smart Insert по структуре всей заметки.
- Режимы производительности как пользовательский тумблер.

Gemma в MVP обязательна, но её текущий слой честно допускает runtime-ошибку. Если LiteRT-LM на конкретной сборке не поднимется, OCR-поток всё равно покажет распознанный текст, а очередь покажет состояние Gemma.

## Первый запуск

При старте приложение проверяет доступность OCR-модуля ML Kit Text Recognition через Google Play Services и наличие файла Gemma.

Если чего-то нет, пользователь видит короткий экран с кнопкой скачивания. Кнопка запускает официальный `ModuleInstallClient` для OCR и скачивает Gemma 4 E2B SM8750 с Hugging Face.

Модель хранится отдельно от APK во внутреннем хранилище приложения и не скачивается заново при обычном `installDebug`.

## Smart Insert в MVP

Сейчас Smart Insert простой:

- если заметка пустая, добавляется первый блок;
- если есть активный текст, новый блок добавляется как продолжение;
- повторы отсекаются по похожему тексту;
- каждый блок остаётся редактируемым в поле заметки.

Позже этот слой станет отдельным Note State Engine с LLM-подсказкой и структурной вставкой.
