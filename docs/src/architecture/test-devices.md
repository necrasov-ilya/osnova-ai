# Тестовые устройства

Эта страница фиксирует реальные устройства, на которых проверяем OSNOVA AI. Синтетические оценки полезны, но решение по pipeline надо принимать после живых прогонов.

## Основное устройство

Первое целевое устройство для тестов:

```text
Vivo X200 Ultra
SoC: Snapdragon 8 Elite
```

На нём проверяем:

- запуск CameraX preview и `ImageAnalysis`;
- стабильность sampler;
- скорость ML Kit Text Recognition;
- время первого запуска Gemma E2B через LiteRT-LM;
- скорость ответа VLM на кроп доски, слайда и экрана;
- нагрев при активной камере;
- поведение GPU/NPU backend;
- расход батареи за длинную учебную сессию;
- качество Smart Insert в реальной заметке.

## Как измеряем

Для каждого теста фиксируем:

- версию приложения;
- версию модели;
- backend: CPU, GPU или NPU;
- режим питания Android;
- заряд на старте;
- длительность сессии;
- среднее время OCR;
- среднее время VLM/LLM;
- время до первого полезного блока;
- длину очереди задач;
- thermal status, если доступен;
- субъективную оценку: мешает ли задержка во время пары.

## Правило

Первые выводы не переносим на все Android-устройства. Vivo X200 Ultra нужен как сильная точка старта. После него понадобится хотя бы одно среднее устройство, чтобы понять реальный нижний порог.
