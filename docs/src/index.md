# OSNOVA AI

OSNOVA AI - Android-приложение для live-генерации конспектов из визуального потока камеры.

Пользователь наводит камеру на доску, слайд, тетрадь, экран или учебный материал. Приложение понимает, что находится в кадре, выбирает маршрут обработки и постепенно дополняет текущую заметку структурированным конспектом.

Главная мысль: это не отдельная камера и не отдельный чат. Это редактор заметок, который умеет смотреть на материал вместе с пользователем.

## Как ведём эту документацию

Документация живёт рядом с проектом и постепенно собирает наши решения. Сырые идеи сначала попадают в журнал, потом дозревают до отдельных страниц.

Если для решения нужна внешняя информация, фиксируем короткий вывод и ссылки на источники. Не надо превращать страницу в конспект интернета.
